# EvalLense Landing — Knowledge Vault

Главный вход в документацию проекта.

Слои проекта:

`notes -> wiki -> Linear -> src`

- `notes/` — сырой capture: мысли, баги, идеи, дневные отчёты
- `wiki/` — документация: product, architecture, stories, decisions, processes
- `Linear` — execution tracking (опционально, см. [[linear-workflow|Linear Workflow]])
- `src/` — реализация

Карта слоёв, статусы документов и рабочий ритм — в [[process|Process]].

## Куда идти дальше

- [[index|Wiki Index]] — навигация по wiki
- [[process|Process]] — слои, документы, workflow
- [[agent-workflow|Agent Workflow]] — агентский контур разработки
- [[linear-workflow|Linear Workflow]] — работа с Linear (если подключён)

## Важное правило

Базовый путь — `notes -> wiki -> Linear -> src`, но это не жёсткая цепочка: задача может идти напрямую `notes -> Linear`, если контекст ещё не дозрел до полноценной документации. Подробнее — в [[process|Process]].
