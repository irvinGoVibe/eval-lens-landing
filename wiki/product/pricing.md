## Роль и аудитория

- **Роль страницы:** показать модель монетизации и снять ценовые возражения —
  EvalLense оплачивается **за мероприятие** (пакет заявок + срок действия), а не
  per-seat и не помесячно. Дать сравнить тарифы между собой и с альтернативными
  способами оценки.
- **Для кого:** организаторы, принимающие решение о покупке — VC-фонды,
  акселераторы, конкурсы, хакатоны, грантовые программы, вузы.
- **Ключевое сообщение:** платите за мероприятие, а не за места; цена покрывает
  **AI-оценку как интеллект**, а не просто приём заявок — поэтому дороже
  event-ops инструментов, но прозрачнее и дешевле зрелых awards-платформ.
- **Целевое действие:** Book a Demo → `/company/contact`; вторичное — Start Free
  (бесплатная проба, 3 заявки).

## Контент по секциям

### 2. Тарифы (4 карточки)
4 headline-карточки (цифры из финмодели, лист «Тарифы» v0.4):

| Карточка | Цена | Заявок | Срок | Роль | CTA |
|---|---|---|---|---|---|
| **Бесплатная проба** | $0 | 3 | до использования | активация продукта | Start Free |
| **Стандартный** ⭐ *recommended* | $1 500 | 150 | 120 дней | основная выручка | Get Started |
| **Профессиональный** | $3 900 | 500 | 180 дней | масштабирование | Get Started |

- Выделить **Стандартный** как рекомендуемый (lens-бордер/бейдж).

- Под карточками — узкая строка **smaller plans**: «Меньше мероприятие? Стартовый
  $199 / 15 заявок · Пилотный $500 / 40 заявок — см. полное сравнение ниже».
  Эти два уровня живут в матрице (секция 3), а не в headline-карточках.
- Под каждой карточкой — сноска про доп-пакеты (см. секцию 3 / FAQ).

### 3. Что входит в тариф (матрица фич × тариф)
Полная таблица: строки-фичи, колонки — **все 6 уровней** (Free · Стартовый ·
Пилотный · Стандартный · Профессиональный · Корпоративный). Здесь покупатель
видит Стартовый/Пилотный, которых нет в headline-карточках. На мобайле —
горизонтальный скролл / аккордеон. Группы строк (черновик, факты — из
product-доков и модели):
- **Лимиты:** заявок включено (3 / 15 / 40 / 150 / 500 / 1000), срок действия
  (до использования / 90 / 90 / 120 / 180 дн / по договору).
- **Доп-пакеты заявок:** — / +$150·10 / +$150·10 / +$300·25 / +$500·50 /
  по договору (цена за заявку в пакете: $15 / $15 / $12 / $10).
- **Оценка:** 6 AI-судей (P1–P6), evidence-привязка, human final scoring —
  **на всех тарифах** (это бренд-обещание, не дифференциатор).
- **Отчёты:** evidence-репорт на участника; export CSV/PDF *(P1)*.
- **Review:** Review Board + leaderboard — на всех тарифах.
- **Admin / Pro+:** кастомные веса критериев, шаблоны мероприятий, бандл-экспорт,
  базовый брендинг *(P1)*; SSO/SAML, data residency, API — Корпоративный.
- **Поддержка:** базовая → приоритетная → выделенная (Корпоративный).

> Принцип: **безопасность и доверие (prompt-injection safety, privacy,
> evidence-linked scoring, human-in-the-loop) — на всех тарифах, включая
> бесплатную пробу.** Дифференцируем объёмом и admin-возможностями, не доверием.

### 4. EvalLense vs альтернативы
**Named-матрица наличия фич, БЕЗ чужих цен.** Колонки — конкуренты + EvalLense;
строки — наши фичи. Смысл: каждый конкурент закрывает только часть, **EvalLense —
единственная колонка со сплошными ✓** (мы = объединение «workflow-платформ» и
«AI-аналитиков деков»). Колонки (из competitor-landscape.md): **YouNoodle**
(competition workflow), **Evalato** (awards/applications), **PitchBob VC** (AI deck
analyst), **EvalLense**.

| Фича | YouNoodle | Evalato | PitchBob VC | **EvalLense** |
|---|---|---|---|---|
| AI-оценка деков как ядро | ✗ | ✗ | ✓ | ✅ |
| Методология 6 независимых судей (P1–P6) | ✗ | ✗ | частично | ✅ |
| Прозрачный rationale (no black box) | ✗ | ✗ | частично | ✅ |
| Evidence-привязка scoring | ✗ | ✗ | частично | ✅ |
| Оценка всего мероприятия (не один дек) | ✓ | ✓ | ✗ | ✅ |
| Leaderboard / ранжирование | ✓ | ✓ | ✗ | ✅ |
| Human-in-the-loop финальное решение | ручное | ручное | ✗ | ✅ |
| Вопросы для живого жюри | ✗ | ✗ | ✗ | ✅ |

*(значения — черновик из feature-матриц research §5/§8; верифицировать факты по
фичам конкурентов перед публикацией.)*

Positioning-строки под таблицей (готовые, из research §8):
- vs workflow-платформы: **«They manage the competition. We help understand who
  should win and why.»**
- vs AI-аналитики деков: **«They review decks. We run structured evaluation
  across an entire competition.»**

> ⚠️ Цены конкурентов на странице НЕ показываем (решение user) — это снимает риск
> устаревания цен. Остаётся верифицировать **фактические утверждения о фичах**
> (есть/нет) — research draft v0.2.

### 5. Возможности по категориям (сетка)
Bento-сетка фич, сгруппированная по продуктовым модулям (это и есть «список
фич»; факты — из [[overview|Обзор продукта]] и trust-страниц):
- **Intake** — приём деков (PDF/PPTX/Slides), self-upload по ссылке, batch.
- **AI Evaluation** — 6 независимых судей P1–P6, confidence, evidence-привязка.
- **Reports** — explainable-отчёт на участника, strengths/weaknesses, экспорт.
- **Review & Decision** — Review Board, human final scoring, leaderboard.
- **Security & Trust** — prompt-injection safety, privacy, no-black-box rationale.
- **Admin** — кастомные веса, шаблоны мероприятий, брендинг, SSO/API (по тарифу).

### 6. FAQ по прайсингу
Черновик вопросов:
- **Что считается «заявкой»?** — одна оценённая презентация/дек.
- **Срок действия тарифа?** — 90/120/180 дней; Корпоративный — по договору.
- **Что если заявок не хватит?** — доп-пакеты (от $150/10, дешевле на старших
  тарифах); либо переход на тариф выше.
- **Это подписка?** — нет, оплата за мероприятие (пакет + срок), не помесячно.
- **Есть ли платный пилот?** — да, Пилотный $500 / 40 заявок → Talk to Sales.
- **Безопасны ли мои данные?** — да, на всех тарифах (→ `/trust/security-privacy`).
- **Кто принимает финальное решение?** — человек; AI готовит анализ (→ `/trust`).

### 7. Final CTA
- Призыв: запустить оценку на своём мероприятии.
- Кнопки: Book a Demo · Start Free.
