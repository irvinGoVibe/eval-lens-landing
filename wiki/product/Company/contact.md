# Contact

## Page purpose

Provide a simple path to start a conversation with EvalLense.

## User task

Help visitors quickly choose the right contact reason: demo, pilot, partnership, support, or general inquiry.

## Main idea to communicate

It should be easy to start a pilot, request a demo, or talk to the team.

## Section structure

### 1. Hero

Purpose:

```text
Make the contact path clear and low-friction.
```

Message:

```text
Talk to EvalLense about demos, pilots, partnerships, or startup evaluation workflows.
```

### 2. Book a Demo

Purpose:

```text
Primary conversion path.
```

Content:

```text
For funds, accelerators, competitions, and programs that want to review pitch decks faster.
```

CTA:

```text
Book a Demo
```

### 3. Pilot request

Purpose:

```text
Capture users who want to test EvalLense on a real batch.
```

Content:

```text
Tell us about your batch size, review process, timeline, and desired outputs.
```

### 4. Partnership

Purpose:

```text
Capture ecosystem, accelerator, and platform partnerships.
```

Content:

```text
For organizations that want to integrate structured startup evaluation into their program or ecosystem.
```

### 5. General contact

Purpose:

```text
Handle everything else.
```

Content:

```text
Questions, media, hiring, or other inquiries.
```

### 6. Response expectation

Purpose:

```text
Set a clear expectation.
```

Message:

```text
The team will review the request and follow up if there is a relevant fit.
```
